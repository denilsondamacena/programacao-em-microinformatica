## Atividade Final


### Exercício Final 

>Desenvolva um sistema de gerenciamento de biblioteca digital que integre todos os conceitos fundamentais do Python:
> ```
>a) Range: para paginação, numeração e iterações controladas
>b) For: para percorrer coleções e processar dados
>c) Funções: para modularizar o código e reutilizar funcionalidades
>d) Arquivos: para persistir dados em JSON e gerar relatórios
>e) Tuplas: para armazenar dados imutáveis como coordenadas e configurações
>f) Dicionários: para estruturar informações complexas de livros e usuários
> ```
>
>O sistema deve permitir:<br>
> ``` 
>g) Cadastro e gerenciamento de livros
>h) Cadastro e gerenciamento de usuários
>i) Sistema de empréstimo e devoluções
>j) Busca avançada com filtros
>k) Relatórios e estatísticas 
>l) Persistência de dados em arquivos
>m) Interface interativa com menus paginados
> ```
