### Exercício 1 - Filtrar Primos
>Crie uma função que receba uma lista de valores (que podem ser números, strings ou
outros tipos) e retorne uma nova lista contendo apenas os números primos válidos. A
função deve tratar exceções para valores inválidos e usar estruturas condicionais para
validação.
Requisitos técnicos:
> ```
>a) Use try-except para tratar conversões inválidas
>b) Implemente verificação de primo com for e range
>c) Use if-elif-else para diferentes casos de validação
>d) A função deve aceitar qualquer iterável como entrada
> ```
>
>Exemplo:<br>
>**Entrada = [2, 3, \"4\", 5, \"abc\", 7.0, 11, \"13\", None, 17]**<br>
>**Saida_esperada = [2, 3, 5, 7, 11, 13, 17]**


### Exercício 2 - Sistema de Login
>Desenvolva um sistema de autenticação que permita ao usuário fazer login com
tentativas limitadas. O sistema deve ter diferentes níveis de usuário e implementar
bloqueio temporário após falhas consecutivas.
Requisitos técnicos:
> ```
>a) Use while para controlar o loop de tentativas
>b) Implemente try-except para tratamento de entrada
>c) Use if-elif-else para diferentes níveis de acesso
>d) Use range para controlar tempo de bloqueio
>e) Crie função para validação de credenciais
> ```

### Exercício 3 - Análise de Vendas
>Crie um sistema que analise dados de vendas de uma empresa, calculando
estatísticas por período, vendedor e região. O sistema deve tratar dados inconsistentes e
gerar relatórios detalhados.
Requisitos técnicos:
> ```
>a) Use for com range para iterar por períodos
>b) Implemente try-except para tratar dados inválidos
>c) Use if-elif-else para categorização de desempenho
>d) Crie função para cálculos estatísticos
>e) Use while para menu interativo
> ```

### Exercício 4 - Gerador de Senhas Avançado
>Desenvolva um gerador de senhas que crie senhas seguras baseadas em critérios
específicos, com validação de força e opções de personalização avançadas.
Requisitos técnicos:
> ```
>a) Use range para controlar comprimento e iterações
>b) Implemente try-except para validação de entrada
>c) Use if-elif-else para diferentes níveis de segurança
>d) Crie função para validação de força da senha
>e) Use while para regeneração até atingir critérios
> ```

### Exercício 5 - Calculadora de Impostos
>Desenvolva uma calculadora que calcule impostos progressivos baseados em
diferentes faixas de renda, com simulações de cenários e comparações anuais.
Requisitos técnicos:
> ```
>a) Use range para iterar pelas faixas de imposto
>b) Implemente try-except para validação de valores monetários
>c) Use if-elif-else para aplicar diferentes alíquotas
>d) Crie função para cálculos de cada faixa
>e) Use while para simulações múltiplas
> ```
